0.6.1. Features• Arithmetic operators: +, -, *, /, %• Comparison operators: ==, !=, >, <, >=, <=• Ternary operator: condition ? true : false• Map transformations with conditional logic• String concatenation in expressions

0.6.2. Examples

`jsson
users [
  template { name, age }

  map (u) = {
    name = u.name
    age = u.age
    isAdult = u.age >= 18
    category = u.age >= 18 ? "adult" : "minor"
  }

  "João", 25
  "Maria", 16
]
`
